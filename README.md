# CHIKV Thermal Functions

This project collects and compares temperature-dependent thermal response functions used in chikungunya transmission risk models.

## Project Structure

- `analysis/`: R Markdown analysis files.
- `scripts/`: Reusable R scripts and checks.
- `figures/`: Generated thermal-response plots.
- `output/`: Rendered reports and generated outputs.
- `docs/`: Project notes, thermal-function tables, and source documents.

## Main Files

- `analysis/Temp_functions.Rmd`: Current thermal-function plots and checks.
- `scripts/check_consistency.R`: Helper script for extracting the latest thermal-function table.
